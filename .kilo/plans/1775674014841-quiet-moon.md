### 4. No Input Validation
- All API endpoints accept raw input

- Auth endpoints vulnerable to brute force
- No protection against DDoS

### 6. Hardcoded JWT Secret Fallback
- `routes/auth.js`: Falls back to predictable secret
- Should fail if env var missing in production

### 7. CORS Too Permissive
